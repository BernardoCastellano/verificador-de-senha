Como foi feito:

HTML:
Quando criei o HTML primeiro eu liguei o CSS que fiz separado com o "head" do meu HTML, após isso, criei uma "section" para guardar a "div" e para depois conseguir posiciona-la na página, depois eu criei a "div", onde criei três elementos "span" e dentro de cada um criei um elemento "label" para dar um nome ao outro elemento, um "input" onde seria escrito o texto pelo usuário. Dentro desta "div" eu criei outra divisão para dois "button", um que verifica a senha, e outro que leva até esta página. No final, inclui um elemento "script", que utilizei para ligar o JavaScript com o documento HTML.


JavaScript:
Agora, partindo do "HTML" para o JavaScript, foi me passado a atividade de criar um programa que verifica se uma senha tem pelo menos 8 caracteres e se contém no mínimo 1 caractere especial, e se alguma destas condições não se cumprisse o erro deveria ser exposto em formato de "alert". Eu programei ele riando uma função chamada "verifyPassword()", dentro dela haveria duas variáveis para coletar os dados da senha e por-los dentro das variáveis e uma variável chamada "erro" que guarda uma "array" vazia, que srá usada posteriormente. Para fazer a filtragem eu utilizei "if" onde o parâmetro passado era a variável da senha mais o método ".match()", que funciona pegando a "string" que foi passado a ele e procura um valor posto nele como parâmetro. No fim, declarei o botão de verificação em uma variável "btnVerify" e introduzi a função a ele.
